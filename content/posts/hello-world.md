This is the first post on the new blog, built with [Hugo](https://gohugo.io/) and
deployed to GitHub Pages via GitHub Actions.

## Writing a new post

```bash
hugo new content posts/my-new-post.md
```

Edit the file, flip `draft = false` in the front matter, then push to `master` —
the Actions workflow builds and publishes automatically.

## What's next

The old site has been archived under `old/`. Everything here is a clean, themeless
Hugo setup that's easy to extend.
